# Poly Multisample Builder test v10

Windows release build for the current Nymphs-Fork Poly Multisample Builder state. This is a normal release, not a prerelease.

## Asset

- `nt_helper-windows-poly-multisample-builder-test-v10.zip`

## Changes

- Manual Decent edits no longer shift, reseed, swap, or prune rows.
- Manual mode warns and blocks Continue on impossible overlaps.
- Chromatic, Velocity layers, Round robins, and Add unmapped export the full visible Low/Root/Vel/RR row map.
- Use Decent map remains XML-preserving and shows XML-derived row summaries until overridden.
- Decent tag/group header alignment and warning spacing were tightened.
